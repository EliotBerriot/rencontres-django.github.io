Notifications asynchrones en Django HTML MP4

Les attentes d'application web avec temps réel sont de plus en plus diffuses. Le développement asynchrone apporte parfois quelques difficultés, Django ne supporte pas l'asynchrone et c'est limitant pour les applications temps réel qui entrent sur des sites de production.

Il y a de nombreuses solutions, Disqus utilise flasq, certains sortent du monde python au profit de node.js...

Nous allons voir comment quelques lignes de Tornado (micro framework développé par FriendFeed et publié par Facebook lors du rachat) peuvent être une bonne solution et comment l'intégrer avec une application Django.

Christophe Narbonne

Étudiant et stagiaire, bidouilleur python et Django passionné. "Si je rêve seul c'est seulement un rêve, mais si nous rêvons ensemble alors c'est le commencement de la réalité."

MySmeuh : plateforme sociale libre HTML MP4

MySmeuh est une plateforme web développée et hebergée par l'association Smeuh.Org. Basée sur Django et Pinax, elle permet à ses utilisateurs de partager leurs photos, leurs créations musicales et leur prose.

Développeur Python et web. Co-fondateur de l'association Smeuh.Org et principal développeur de MySmeuh.

Neige Freu HTML MP4

Les anglais avaient uksnow. une riche idée qui crowdesource les averses de neige via un simple hashtag twitter.

Jaloux comme un poney, Bruno Bord a décidé l'hiver dernier de faire neigefr, sur le même principe. En Django.

Y'a pas de raison que les bifs soient les seuls à couvrir leur carte de neige virtuelle.

Bruno Bord

Développeur web curieux depuis l'an 2000, Bruno a suivi une progression logique dans son métier : de l'ASP moche, du PHP from scratch, des CMS overkill, pour finalement se rendre compte qu'on pouvait résoudre les problèmes sans avoir trop mal à la tête : en utilisant Python et Django.
Après, les problèmes qu'il résout, on se demande des fois si ça valait le coup de les résoudre. Mais bon.

Ionyweb, un CMS Django pour les poneys HTML MP4

Ionyweb, c'est l’aboutissement de 3 ans de R&D pour démarrer un site en un clin d'œil et l'adapter aux besoins des clients tout en leur permettant de gérer simplement le contenu de leur site.

Basé sur une API REST et une IHM javascript qui se greffe sur le site lors de la connexion, Ionyweb intégre la gestion des thèmes, des applications de pages, des plugins, des layouts ainsi qu'un gestionnaire de fichiers.

Aujourd'hui Ionyweb se dévoile.

Rémy Hubscher

Rémy HUBSCHER découvre Django en 2007. En 2009, il monte Ionyse avec deux amis pour vivre de ce framework et le déployer chez ses clients.
Durant ces trois années, Ionyse développe plusieurs projets dont django-intranet une brique de base pour réaliser des intranets flexibles pour les entreprises.
Aujourd'hui Ionyse dévoile Ionyweb, un CMS basé sur une API.
