Ionyweb, un CMS Django pour les poneys

Ionyweb, c'est l’aboutissement de 3 ans de R&D pour démarrer un site en un clin d'œil et l'adapter aux besoins des clients tout en leur permettant de gérer simplement le contenu de leur site.

Basé sur une API REST et une IHM javascript qui se greffe sur le site lors de la connexion, Ionyweb intégre la gestion des thèmes, des applications de pages, des plugins, des layouts ainsi qu'un gestionnaire de fichiers.

Aujourd'hui Ionyweb se dévoile.

Public
Débutant/Débrouillard
Pré-requis
Un environnement de développement Django opérationnel.

Rémy Hubscher

Rémy HUBSCHER découvre Django en 2007. En 2009, il monte Ionyse avec deux amis pour vivre de ce framework et le déployer chez ses clients.
Durant ces trois années, Ionyse développe plusieurs projets dont django-intranet une brique de base pour réaliser des intranets flexibles pour les entreprises.
Aujourd'hui Ionyse dévoile Ionyweb, un CMS basé sur une API.
